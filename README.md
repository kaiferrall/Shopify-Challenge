# Image-Gallery
Customize an image gallery and save it for viewing.

Local Usage:

  1. download/clone
  2. npm install in root and in client
  3. run node server.js from root
  4. run npm start
  
Find the actual site here: https://myimagegallery.herokuapp.com/

  Technologies Used:
   1. React.js
   2. Node.js
   3. MongoDB
   4. CSS/HTML/Javascript
